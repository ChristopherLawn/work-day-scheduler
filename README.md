# Work Day Scheduler

## Purpose
A simple, easy-to-use daily task scheduler! It conveniently shows the user the current day and time and color-codes each hour to let the user know which tasks are overdue and which tasks are coming up!

## Built With
* HTML
* CSS
* JS
